# HackMan Api

## Documentation

* Method: GET
* Endpoint: http://clemsonhackman.com/api/word
* Query String Params:
  * Key (Required): The api key you were given
  * Length (Optional): A number between 4 and 15 that denotes the length of the word for the api to return

* Returns:
  * Json
  * 
```
    {
      "word": "yourword"
    }
```
